Sobre o Sports Portal

Um portal informativo sobre desportos

Sobre o site

Este site usa várias API's para obter informações sobre desportos como Formula 1 e Counter-Strike: Global Offensive. Além dessas API's também são usadas outras para complementar para oferecer uma informação mais completa. Optou-se na qualidade de uso em vez da quantidade de API's. Todas elas são muito completas, com muita informação a cada pedido que é feito, o que faz com que não fossem preciso outras API's para obter dados extra.

API's

As API's usadas são:

  • Ergast F1
  • PandaScore
  • NewsAPI
  • Aeris Weather

Tecnologias

Para a estrutura este website é usado simples HTML. Quanto à estilagem, é à base de CSS básico, assim como intermédio, com algumas animações e também importação de alguns templates básicos.

A obtenção da informação e tratamento dos dados das API's é usado JavaScript e a biblioteca jQuery. Através destes dois, foi possivel fazer tudo que é visto no website inteiro.
